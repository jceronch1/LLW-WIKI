# Copilot Studio

**Tipo:** programa

## Descripción
Plataforma de Microsoft para la creación de agentes personalizados con enfoque de bajo código (low-code), integrada con Microsoft 365 y Azure.

## Rol en el ecosistema de IA
Herramienta empresarial para la creación y despliegue rápido de agentes de productividad.
