# Agent Skills

**Tipo:** norma

## Descripción
Especificación de estándar abierto para habilidades portátiles de agentes de IA, que permite que las capacidades de los agentes sean vistas como bibliotecas de código.

## Rol en el ecosistema de IA
Estandarización de módulos de funcionalidad para agentes de IA, introduciendo consideraciones de seguridad en la cadena de suministro de plugins y habilidades.
