# Milvus

**Tipo:** organizacion

## Descripción
Proyecto de base de datos vectorial de código abierto cuya documentación se utiliza como referencia técnica para la complejidad de los algoritmos de búsqueda del vecino más cercano aproximado (ANN) como LSH, HNSW y IVF.

## Rol en el ecosistema de IA
Proveedor de infraestructura de búsqueda vectorial documentada para sistemas de IA y RAG.
