# CaMeL

**Tipo:** programa

## Descripción
Marco de trabajo (framework) de seguridad para agentes de IA que separa explícitamente el flujo de control del flujo de datos para defender contra inyecciones de prompt indirectas.

## Rol en el ecosistema de IA
Arquitectura de defensa a nivel de sistema que utiliza un LLM privilegiado y un LLM en cuarentena, junto con un sistema de seguimiento de flujo de datos basado en capacidades.
