# Google Agent Builder

**Tipo:** programa

## Descripción
Plataforma de Google Cloud que facilita la creación de agentes de búsqueda y conversación utilizando Vertex AI.

## Rol en el ecosistema de IA
Herramienta en la nube para la construcción de agentes conversacionales y de búsqueda empresarial.
