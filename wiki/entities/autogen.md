# AutoGen

**Tipo:** programa

## Descripción
Framework de Microsoft basado en conversaciones que permite el diálogo entre múltiples agentes y la ejecución de código, altamente personalizable.

## Rol en el ecosistema de IA
Herramienta de desarrollo para sistemas multiagente conversacionales y ejecución de tareas.
