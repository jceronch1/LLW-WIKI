# CrewAI

**Tipo:** programa

## Descripción
Framework de código abierto para agentes de IA basado en roles, que facilita la colaboración entre agentes mediante procesos secuenciales y jerárquicos.

## Rol en el ecosistema de IA
Herramienta de desarrollo para sistemas multiagente colaborativos.
