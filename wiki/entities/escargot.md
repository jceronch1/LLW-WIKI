# ESCARGOT

**Tipo:** programa

## Descripción
Agente de IA que integra modelos de lenguaje grandes, grafos de pensamiento dinámicos y grafos de conocimiento biomédico para razonamiento mejorado en bioinformática.

## Rol en el ecosistema de IA
Sistema de agente único con capacidades de auto-depuración de código Python y razonamiento sobre grafos de conocimiento para análisis complejos de datos biológicos.
