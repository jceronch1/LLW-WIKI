# software-clinismo-confinado-predictible

## Definición
Evolución de los sistemas clínicos (CCS) que utilizan técnicas como el aprendizaje profundo para manejar relaciones IDOL con vínculos desconocidos pero dentro de un espectro de salida limitado. Aunque presentan variabilidad, su comportamiento permanece dentro de límites controlados y predecibles.

## Ideas clave
- Uso de deep learning
- Variabilidad predecible
- Requiere guardrails para seguridad
- Escala dentro de marcos SaMD existentes
