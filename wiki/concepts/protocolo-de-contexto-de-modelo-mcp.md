# protocolo-de-contexto-de-modelo-mcp

## Definición
Estándar abierto introducido por Anthropic que proporciona una interfaz universal para que los modelos de inteligencia artificial se conecten a fuentes de datos externas y herramientas de software. Resuelve el problema de la integración combinatoria al estandarizar la conexión de múltiples modelos con múltiples herramientas, sirviendo como infraestructura clave para el despliegue de sistemas multiagente.

## Ideas clave
- Estándar abierto para conexión de IA con datos y herramientas
- Resuelve el problema de integración combinatoria
- Infraestructura clave para sistemas multiagente
- Rápida adopción por OpenAI, Google DeepMind y Microsoft
