# Control de acceso adaptativo al riesgo para agentes

## Definición
Modelo de autorización que combina el control de acceso basado en roles (RBAC) con limitaciones cuantitativas de riesgo, donde las decisiones de autorización están sujetas a límites sobre riesgos agregados. Este enfoque híbrido ofrece una base práctica para políticas de autorización deterministas en entornos de agentes dinámicos, donde el RBAC por sí solo resulta insuficiente.

## Ideas clave
- Combina RBAC con enfoques de control de acceso adaptativo al riesgo
- Las decisiones de autorización están sujetas a limitaciones de riesgos agregados
- RBAC por sí solo es insuficiente para entornos de agentes dinámicos
- Proporciona una base práctica para políticas de autorización deterministas
