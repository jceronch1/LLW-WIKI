# memoria-a-largo-plazo-en-agentes

## Definición
Capacidad de un agente de IA para almacenar y recuperar información entre diferentes sesiones de interacción. Es crucial para mantener el contexto, permitir la personalización y asegurar la continuidad en la ejecución de tareas complejas a lo largo del tiempo.

## Ideas clave
- Retención de contexto entre sesiones
- Fundamental para la personalización
- Permite aprendizaje de experiencias pasadas
- Esencial para la continuidad de tareas complejas
