# sismografia-de-senal-metrica-d0

## Definición
Método de detección de amenazas que monitorea en tiempo real la dimensión fractal de Rényi (D₀) de las distribuciones de probabilidad de salida del modelo, tratando la seguridad como un problema de procesamiento de señales. Las perturbaciones adversarias "fracturan" la curva de probabilidad natural, produciendo valores D₀ anómalos que preceden a las acciones dañinas.

## Ideas clave
- Transforma la monitorización de seguridad de análisis de contenido a verificación estructural pre-acción
- Permite el enclavamiento de ejecución (execution gating) para acciones de alta consecuencia
- Establece líneas base de D₀ bajo condiciones operativas seguras para comparación
