# agente-l-pentai

## Definición
Sistema de IA autónomo y local diseñado para realizar pruebas de penetración internas de manera continua, combinando aprendizaje por refuerzo (RL) y generación aumentada por recuperación (RAG) para descubrir vulnerabilidades y configuraciones incorrectas. El agente observa el estado de la red interna, recupera contexto de seguridad relevante y selecciona acciones para maximizar la recompensa de encontrar fallos de seguridad.

## Ideas clave
- Operación local sin envío de datos sensibles externos
- Uso de RL para exploración del entorno de red
- Arquitectura multiagente coordinada por un planificador LLM
- Búsqueda de vecinos aproximados para recuperación eficiente
- Formalización del entorno como un Proceso de Decisión de Markov (MDP)
